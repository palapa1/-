# 软件公司售后服务系统

​		为提高软件公司内部售后维护团队信息化，同时为了给客户提供更好的售后服务，方便企业对客户的维护信息有更好的了解。系统主要完成企业客户提交问题的解决、公司任务的分配、任务的记录、问题的查询等功能。
